Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

1. Зайти в директорию, где находится проект/файлы и будет создаваться Git репозиторий
2. команда git status - отобразит состояние, что есть неотсележиваемые 
3. команда git add имя файла
или команда git add .   добавятся все новые файлы в репозитори
4. команда git status - отобразит состояние, добавленные гит директории или файла должны быть зелёными, перейти в состояния new file
5. команда git commit -m "текст коментария как subject на емйле" 
6. команда git status
******    команды на github    *******
1. создать новый репозиторий 
2. скопировать команды с Github по протокол SSН и ввести в терминал

Какую информацию нам дают команда :
git status Команда отображает состояние и позволяет увидеть, какие изменения были подготовлены, а какие нет и какие файлы не отслеживаются Git.
git log Показывает историю, что происходило с репозиторием в обратном порядке, внизу будет 	самый старый коммит и сверху последний
git remote -v  Показывает связку с удаленным репозиторием
